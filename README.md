# printf

Своя упрощенная реализация printf. Поддерживает флаги: [xXcsidup] и спецификаторы: [-0.] (точность, ширина, выравнивание, дополнение нулями).
При сборке проекта создается статическая библиотека, в которой есть printf. Сборка проходит через makefile. <br/>

#### Makefile:
- make
- make clean
- make fclean
- make re
